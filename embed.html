Embed Code for Toast Notifications

Copy and paste the following code snippet into your website:

        
<script defer data-domain="assistable.ai" src="https://assistable-ai.github.io/toasty/script.js"></script>

To display a notification, call the showToast function:

        
<script>
  showToast('This is a toast notification!', 'icon.png', 'top-right', 5);
</script>
